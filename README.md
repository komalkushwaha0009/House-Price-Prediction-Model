# 🏠 House Price Prediction Model

## 📌 Project Overview
This project focuses on predicting house prices using **Multiple Linear Regression (MLR)**.  
It analyzes various structural and geographical features to estimate the **fair market value** of residential properties.


## 🎯 Objective
The main goal of this project is to:
- Build a reliable model for house price prediction
- Reduce manual effort in real estate valuation
- Provide data-driven insights for buyers and sellers


## 📊 Dataset Description
The dataset contains multiple features influencing house prices.

### 🔹 Target Variable:
- `house_price` → Price of the property

### 🔹 Features Include:
- Area / Size
- Number of rooms
- Location factors
- Other structural attributes


## ⚙️ Project Workflow

### 1️⃣ Data Preprocessing
- Handling missing values (Mean Imputation)
- Outlier detection using IQR
- Data cleaning and formatting

### 2️⃣ Exploratory Data Analysis (EDA)
- Data visualization using graphs
- Feature relationships analysis

### 3️⃣ Model Building
- Applied **Multiple Linear Regression**
- Trained model on dataset

### 4️⃣ Model Evaluation
- Performance metrics used:
  - R² Score
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)


## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn


## 📈 Key Insights
- Certain features significantly impact house prices
- Data preprocessing improves model accuracy
- Linear regression provides a good baseline model
